# Responsive Fixed Logo Webpage

![License](https://img.shields.io/badge/License-MIT-blue.svg)

## Description

This project is a Porfolio webpage with animation. It features a visually appealing radial gradient background, smooth animations, and interactive elements like buttons and modals. Designed with modern CSS techniques, it ensures compatibility and a seamless user experience across various devices.

---

## Features

- **Responsive Design**: Logo scales dynamically to fit different screen sizes.
- **Fixed Logo**: The logo remains in a fixed position while scrolling.
- **Interactive Animations**: Smooth animations for title reveal, button hover effects, and modal transitions.
- **Modular Structure**: Reusable CSS classes for better maintainability.
- **Accessibility**: User-friendly features like `user-select: none` and intuitive controls.
